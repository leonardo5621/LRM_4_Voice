O que são estes modelos acústicos?

Os modelos acústicos utilizados neste projeto são uma forma de guardar as informações sobre a voz de cada locutor, os dados fornecidos na etapa de criação do modelo são usados para adaptá-los para que eles possam reconhecer a voz do usuário. O algoritmo utilizado é o do Modelo de Mistura de Gaussianas(GMM).
